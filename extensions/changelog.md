# Changelog

All notable changes to our OpenCart extensions are documented here.

## Polish Language Pack

### Version 4.1.0.3

**Release Date:** March 2026

#### Added

- Initial release for OpenCart 4.1.x
- Complete Polish translation for admin panel
- Complete Polish translation for storefront
- Payment modules translations (bank transfer, COD, cheque, free checkout)
- Shipping modules translations (flat, weight, item, free, pickup)
- Total modules translations (subtotal, total, handling, low order fee, credit, coupon, reward)
- Module translations (bestseller, featured, latest, special, category, account, information, filter, blog, store, topic)
- Email templates translations
- GDPR compliance translations

#### Compatibility

- OpenCart 4.1.0.3
- OpenCart 4.1.0.2
- OpenCart 4.1.0.1
- OpenCart 4.1.0.0
