## Installation

All language packs can be installed directly through OpenCart's extension installer:

1. Download the language pack from OpenCart Marketplace
2. Go to **Extensions > Installer** in your admin panel
3. Upload the extension file
4. Go to **System > Localization > Languages**
5. Add the new language configuration

For detailed installation instructions, see our [Installation Guide](../getting-started/installation.md).
