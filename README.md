# DESIGN4ᴾᴿᴼ OpenCart Community

Official documentation repository for OpenCart language packs developed by DESIGN4ᴾᴿᴼ.

## Overview

This repository contains documentation and sources for language extensions for OpenCart 4.1.x. We develop professional language packs that enable multilingual configuration for OpenCart-based online stores.

## Language Extensions

### Polish Language Pack (oc_language_pl)

Complete Polish translation for OpenCart 4.1.x admin panel and storefront.

**Features:**
- 100% admin interface translation coverage
- Storefront translation (products, cart, checkout, customer account)
- Regular updates for new OpenCart releases

### Lithuanian Language Pack (oc_language_lt)

Complete Lithuanian translation for OpenCart 4.1.x.
