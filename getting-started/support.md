# Support & Feedback

We welcome bug reports and feature requests. Please use GitHub Discussions to ask questions and get help.

## Get Support

- [GitHub Discussions](https://github.com/design4pro/opencart-community/discussions) - Ask questions and get community help
- [Email Support](mailto:dev@design4.pro) - For direct inquiries

> **Note:** For security vulnerabilities, please do not report via GitHub. Email security concerns directly to dev@design4.pro.

## Before Submitting

1. **Search Existing Discussions**
   - Check if your question has already been answered

2. **Verify the Issue**
   - Confirm the issue exists in the latest version

3. **Gather Information**
   - Collect relevant details about your environment

## Required Information

When asking for help, please include:

### Environment Details

- OpenCart version
- PHP version
- Web server (Apache/Nginx)
- Database version
- Language package version

### Issue Description

- Clear description of the problem
- Steps to reproduce
- Expected vs actual behavior

### Screenshots (if applicable)

- Relevant screenshots
- Error messages
- Console logs

## Other Support Channels

- [GitHub Repository](https://github.com/design4pro/opencart-community) - Source code and contributions
- [Email](mailto:dev@design4.pro) - For direct inquiries
