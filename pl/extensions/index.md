# Rozszerzenia OpenCart

Odkryj naszą kolekcję profesjonalnych rozszerzeń dla platformy e-commerce OpenCart. Wszystkie rozszerzenia są zaprojektowane, aby zwiększyć funkcjonalność Twojego sklepu i są w pełni kompatybilne z OpenCart 4.1.x.

## Przeglądaj według Kategorii

| Kategoria | Opis |
|-----------|------|
| [Pakiety Językowe](language/index.md) | Kompletne pakiety lokalizacji dla sklepów międzynarodowych |
| [Motywy](theme/index.md) | Niestandardowe motywy zoptymalizowane pod kątem konwersji (Wkrótce) |
| [Moduły](modules/index.md) | Wzbogać swój sklep potężnymi modułami (Wkrótce) |
| [Płatności i Wysyłka](payment-shipping/index.md) - Bramki płatności i metody wysyłki (Wkrótce) |

## Polecane Rozszerzenia

### Pakiet Języka Polskiego

- **Kategoria:** Pakiet Językowy
- **Wersja:** 4.1.x
- **Opis:** Kompletna polska lokalizacja dla OpenCart 4.1.x - najbardziej kompleksowe dostępne tłumaczenie polskie
- **Cena:** Bezpłatne
- [Zobacz Dokumentację](language/oc-language-pl.md)
