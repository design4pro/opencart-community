# Wsparcie

Uzyskaj pomoc z pakietami językowymi DESIGN4ᴾᴿᴼ OpenCart.

## Jak uzyskać pomoc

### 1. Sprawdź dokumentację

Przed opublikowaniem sprawdź:

- [Przewodnik instalacji](../getting-started/installation.md)
- [Przewodnik konfiguracji](../getting-started/configuration.md)
- [FAQ](../faq/index.md)
- [Rozwiązywanie problemów](../troubleshooting/index.md)

### 2. Wyszukaj istniejące dyskusje

Twoje pytanie może już mieć odpowiedź!

- Wyszukaj [GitHub Discussions](https://github.com/design4pro/opencart-community/discussions)
- Użyj słów kluczowych takich jak "Polish", "installation", "translation"

### 3. Opublikuj nową dyskusję

Jeśli nie możesz znaleźć odpowiedzi:

1. Idź do [GitHub Discussions](https://github.com/design4pro/opencart-community/discussions)
2. Kliknij **Nowa dyskusja**
3. Wybierz odpowiednią kategorię

## Kategorie dyskusji

### Pytania ogólne

W przypadku ogólnych pytań:

- Jak instalować pakiety językowe
- Pomoc w konfiguracji
- Jak używać funkcji

**Użyj tego dla**: Pytań dotyczących konfiguracji lub użytkowania

### Zgłoszenia błędów

W przypadku problemów z rozszerzeniem:

- Błędy tłumaczeń
- Problemy z instalacją
- Problemy z funkcjonalnością

**Dołącz**:
- Wersję OpenCart
- Wersję pakietu językowego
- Kroki do odtworzenia
- Komunikaty błędów (jeśli występują)

### Pomysły i propozycje funkcji

W przypadku sugestii:

- Nowe propozycje tłumaczeń
- Ulepszenia funkcji
- Dodatkowe prośby o języki

**Dołącz**:
- Jasny opis
- Przypadek użycia/powód
- Przykłady (jeśli dotyczy)

## Przed opublikowaniem

### Podaj informacje

Pomóż nam Ci pomóc, dołączając:

| Informacja | Dlaczego jest ważna |
|------------|---------------------|
| Wersja OpenCart | Różne wersje mogą mieć różne rozwiązania |
| Wersja pakietu językowego | Zapewnia, że rozwiązujemy odpowiednią wersję |
| Wersja PHP | Niektóre problemy są specyficzne dla wersji PHP |
| Nazwa motywu | Niestandardowe motywy mogą wpływać na funkcjonalność |
| Kroki do odtworzenia | Pomaga nam zrozumieć problem |
| Komunikaty błędów | Bezpośrednie wskazówki do problemu |

## Czas odpowiedzi

- **Pytania ogólne**: 24-48 godzin (dni robocze)
- **Zgłoszenia błędów**: 48-72 godziny (dni robocze)
- **Propozycje funkcji**: Przegląd tygodniowy

## Czego nie wspieramy

- Niestandardowych modyfikacji naszych rozszerzeń
- Problemów z integracją z motywami zewnętrznymi
- Podstawowej funkcjonalności OpenCart
- Konfiguracji serwera (skontaktuj się z swoim dostawcą hostingu)

## Inne kanały kontaktu

### Rynek OpenCart

- Zostaw opinię na stronie rozszerzenia
- Skontaktuj się przez wiadomości marketplace

### Wsparcie e-mail

W pilnych sprawach biznesowych:
- Sprawdź stronę rozszerzenia dla danych kontaktowych
