# Litewski pakiet językowy

**Litewski pakiet językowy** od DESIGN4ᴾᴿᴼ zapewnia kompletne litewskie tłumaczenia dla panelu administratora i sklepu OpenCart 4.1.x.

## Przegląd

| Funkcja | Szczegóły |
|---------|-----------|
| **Nazwa rozszerzenia** | Litewski pakiet językowy (oc_language_lt) |
| **Wersja** | 4.1.0 |
| **Wersja OpenCart** | 4.1.x |
| **Twórca** | DESIGN4ᴾᴿᴼ |
| **Licencja** | Własnościowa |

## Co zawiera

### Tłumaczenia panelu administratora

- Pulpit
- Katalog (Kategorie, Produkty, Atrybuty, Opcje, Producenci, Recenzje)
- Rozszerzenia (Instalator, Rozszerzenia, Moduły, Wysyłka, Płatność, Suma zamówienia)
- Sprzedaż (Zamówienia, Zamówienia cykliczne, Zwroty, Klienci, Grupy klientów)
- Marketing (Marketing, Mail, Analityka)
- System (Ustawienia, Użytkownicy, Grupy użytkowników, Lokalizacja, Projekt, Konserwacja)
- Raporty (Sprzedaż, Produkty, Klienci, Marketing)
- Pomoc i wsparcie

### Tłumaczenia sklepu

- Nagłówek i stopka
- Menu nawigacyjne
- Strony produktów
- Strony kategorii
- Koszyk
- Proces zamówienia
- Strony konta
- Formularze kontaktowe
- Strony błędów

## Instalacja

Postępuj zgodnie z [Przewodnikiem instalacji](../getting-started/installation.md), aby zainstalować to rozszerzenie.

## Konfiguracja

Po instalacji skonfiguruj litewski jako język swojego sklepu:

1. Wybierz **System** → **Lokalizacja** → **Języki**
2. Dodaj nowy język z:
   - **Nazwa języka**: Litewski
   - **Kod**: `lt`
   - **Locale**: `lt-lt,lt,lithuanian`
   - **Status**: Włączony
3. Wybierz **System** → **Ustawienia** → zakładka **Lokalizacja**
4. Ustaw domyślny język na Litewski

## Wymagania

- OpenCart 4.1.x
- PHP 8.0 lub nowszy
- Uprawnienia do zapisu dla katalogów językowych

## Funkcje

- **Pełne pokrycie** - 100% ciągów znaków OpenCart przetłumaczone
- **Profesjonalne tłumaczenie** - Recenzowane przez native speakerów
- **Przyjazne dla SEO** - Litewskie URL-e włączone
- **Obsługa walut** - Euro (EUR) skonfigurowane
- **Regularne aktualizacje** - Maintainowane dla nowych wydań OpenCart
