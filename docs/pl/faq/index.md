# Najczęściej zadawane pytania

Odpowiedzi na najczęściej zadawane pytania o naszych pakietach językowych OpenCart.

## Pytania ogólne

### Jakie wersje OpenCart są obsługiwane?

!!! info "Tylko OpenCart 4.1.x"
    Nasze pakiety językowe obsługują **wyłącznie OpenCart 4.1.x**. Nie są kompatybilne z OpenCart 3.x lub wcześniejszymi wersjami.

### Gdzie mogę kupić pakiety językowe?

Pakiety językowe są dostępne na [Rynku OpenCart](https://www.opencart.com/index.php?route=marketplace/extension/info&extension_id=48243).

### Czy tłumaczenia są kompletne?

Tak, zapewniamy kompletne tłumaczenia dla:
- 100% ciągów znaków panelu administratora
- 100% ciągów znaków sklepu
- Wszystkich standardowych modułów OpenCart

### Jak często pakiety językowe są aktualizowane?

Regularnie aktualizujemy pakiety językowe, aby:
- Dodawać nowe tłumaczenia dla aktualizacji OpenCart
- Naprawiać błędy tłumaczeń
- Dodawać brakujące ciągi znaków

## Instalacja

### Czy mogę zainstalować pakiety językowe na działającym sklepie?

Tak, ale zalecamy:
- Testowanie na stronie stagingowej najpierw
- Tworzenie kopii zapasowej bazy danych przed instalacją
- Instalację w okresie niskiego ruchu

### Czy potrzebuję wiedzy technicznej do instalacji?

Wymagana jest podstawowa wiedza:
- Dostęp do panelu administratora OpenCart
- Rozumienie podstawowych ustawień sklepu
- Do standardowej instalacji nie jest wymagane kodowanie

### Czy mogę zainstalować wiele pakietów językowych?

Tak! Możesz zainstalować wiele języków:
1. Zainstaluj pierwszy pakiet językowy
2. Skonfiguruj język w ustawieniach
3. Powtórz dla dodatkowych języków
4. Użyj modułu przełącznika języka, aby wyświetlić wszystkie języki

## Konfiguracja

### Jak ustawić domyślny język?

1. Wybierz **System** → **Ustawienia**
2. **Edytuj** swój sklep
3. Przejdź do zakładki **Lokalizacja**
4. Ustaw preferowany **Język**
5. Kliknij **Zapisz**

### Jak dodać przełącznik języka do mojego sklepu?

1. Wybierz **Projekt** → **Układy**
2. Edytuj swój układ (np. "Domyślny")
3. Dodaj moduł **Język** do żądanej pozycji
4. Zapisz układ

### Czy mogę mieć różne języki dla panelu administratora i sklepu?

Tak! Skonfiguruj oddzielnie:
- **Sklep**: System → Ustawienia → zakładka Lokalizacja
- **Admin**: System → Ustawienia → Twój profil użytkownika

## Tłumaczenia

### Co zrobić, jeśli tłumaczenie brakuje lub jest nieprawidłowe?

1. Sprawdź, czy używasz najnowszej wersji
2. Wyszukaj [GitHub Discussions](https://github.com/design4pro/opencart-community/discussions) pod kątem istniejących zgłoszeń
3. Opublikuj nową dyskusję, jeśli nie znaleziono

### Czy mogę sam edytować tłumaczenia?

Tak, OpenCart pozwala nadpisywać tłumaczenia:
1. Wybierz **Rozszerzenia** → **Języki**
2. Edytuj pakiet językowy
3. Modyfikuj poszczególne ciągi znaków

### Czy pakiety językowe zawierają tłumaczenia walut?

Tak:
- **Polski**: Polski złoty (PLN)
- **Litewski**: Euro (EUR)
