## Installation

Follow the [Installation Guide](../getting-started/installation.md) to install this extension.

## Configuration

After installation, configure Lithuanian as your store language:

1. Go to **System** → **Localization** → **Languages**
2. Add new language with:
   - **Language Name**: Lithuanian
   - **Code**: `lt`
   - **Locale**: `lt-lt,lt,lithuanian`
   - **Status**: Enabled
3. Go to **System** → **Settings** → **Local** tab
4. Set default language to Lithuanian
