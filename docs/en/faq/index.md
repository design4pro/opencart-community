# Frequently Asked Questions

Find answers to common questions about our OpenCart language packs.

## General Questions

### What versions of OpenCart are supported?

!!! info "OpenCart 4.1.x Only"
    Our language packs support **OpenCart 4.1.x** only. They are not compatible with OpenCart 3.x or earlier versions.

### Where can I purchase language packs?

Language packs are available on the [OpenCart Marketplace](https://www.opencart.com/index.php?route=marketplace/extension/info&extension_id=48243).

### Are the translations complete?

Yes, we provide complete translations for:
- 100% of admin panel strings
- 100% of storefront strings
- All standard OpenCart modules

### How often are the language packs updated?

We update language packs regularly to:
- Add new translations for OpenCart updates
- Fix any translation errors
- Add missing strings

## Installation

### Can I install language packs on a live store?

Yes, but we recommend:
- Testing on a staging site first
- Backing up your database before installation
- Installing during low traffic periods

### Do I need technical knowledge to install?

Basic knowledge is required:
- Access to OpenCart Admin panel
- Understanding of basic store settings
- No coding required for standard installation

### Can I install multiple language packs?

Yes! You can install multiple languages:
1. Install the first language pack
2. Configure the language in settings
3. Repeat for additional languages
4. Use the language switcher module to display all languages

## Configuration

### How do I set a default language?

1. Go to **System** → **Settings**
2. **Edit** your store
3. Navigate to the **Local** tab
4. Set your preferred **Language**
5. Click **Save**

### How do I add a language switcher to my store?

1. Go to **Design** → **Layouts**
2. Edit your layout (e.g., "Default")
3. Add the **Language** module to desired position
4. Save the layout

### Can I have different languages for admin and storefront?

Yes! Configure separately:
- **Storefront**: System → Settings → Local tab
- **Admin**: System → Settings → Your user profile

## Translations

### What if a translation is missing or incorrect?

1. Check if you're using the latest version
2. Search [GitHub Discussions](https://github.com/design4pro/opencart-community/discussions) for existing reports
3. Post a new discussion if not found

### Can I edit translations myself?

Yes, OpenCart allows you to override translations:
1. Go to **Extensions** → **Language**
2. Edit your language pack
3. Modify individual strings

### Do language packs include currency translations?

Yes:
- **Polish**: Polish Zloty (PLN)
- **Lithuanian**: Euro (EUR)
