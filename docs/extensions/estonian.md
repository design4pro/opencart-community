## Installation

Follow the [Installation Guide](../getting-started/installation.md) to install this extension.

## Configuration

After installation, configure Estonian as your store language:

1. Go to **System** → **Localization** → **Languages**
2. Add new language with:
   - **Language Name**: Estonian
   - **Code**: `et`
   - **Locale**: `et-ee,et,estonian`
   - **Status**: Enabled
3. Go to **System** → **Settings** → **Local** tab
4. Set default language to Estonian
