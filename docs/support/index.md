# Support

Get help with your DESIGN4ᴾᴿᴼ OpenCart language packs.

## How to Get Help

### 1. Check Documentation

Before posting, please check:

- [Installation Guide](../getting-started/installation.md)
- [Configuration Guide](../getting-started/configuration.md)
- [FAQ](../faq/index.md)
- [Troubleshooting](../troubleshooting/index.md)

### 2. Search Existing Discussions

Your question may already be answered!

- Search [GitHub Discussions](https://github.com/design4pro/opencart-community/discussions)
- Use search with keywords like "Polish", "installation", "translation"

### 3. Post a New Discussion

If you can't find your answer:

1. Go to [GitHub Discussions](https://github.com/design4pro/opencart-community/discussions)
2. Click **New Discussion**
3. Choose the appropriate category

## Discussion Categories

### Q&A

For general questions:

- How to install language packs
- Configuration help
- How to use features

**Use this for**: Questions about setup or usage

### Bug Reports

For issues with the extension:

- Translation errors
- Installation problems
- Functionality issues

**Include**:
- OpenCart version
- Language pack version
- Steps to reproduce
- Error messages (if any)

### Ideas & Feature Requests

For suggestions:

- New translation suggestions
- Feature improvements
- Additional language requests

**Include**:
- Clear description
- Use case/reason
- Examples (if applicable)

## Before Posting

### Provide Information

Help us help you by including:

| Information | Why It's Important |
|-------------|---------------------|
| OpenCart Version | Different versions may have different solutions |
| Language Pack Version | Ensures we're troubleshooting the right version |
| PHP Version | Some issues are PHP version specific |
| Theme Name | Custom themes may affect functionality |
| Steps to Reproduce | Helps us understand the issue |
| Error Messages | Direct clues to the problem |

### Check These First

- [ ] Reviewed the [Installation Guide](../getting-started/installation.md)
- [ ] Reviewed the [FAQ](../faq/index.md)
- [ ] Reviewed the [Troubleshooting Guide](../troubleshooting/index.md)
- [ ] Searched existing discussions
- [ ] Cleared cache and tried again

## Response Times

- **Q&A**: 24-48 hours (business days)
- **Bug Reports**: 48-72 hours (business days)
- **Feature Requests**: Weekly review

## What We Don't Support

- Custom modifications to our extensions
- Third-party theme integration issues
- OpenCart core functionality
- Server configuration (please contact your host)
